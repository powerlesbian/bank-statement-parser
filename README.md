# Bank Statement Parser

A TypeScript tool to parse bank statements from multiple Hong Kong banks (BOC, HSBC, Standard Chartered) and extract transactions for accounting/expense tracking.

## Supported Banks
- Bank of China (BOC)
- HSBC
- Standard Chartered Bank (SCB)

## Features
- PDF statement parsing
- XLSX/CSV support
- Transaction extraction and normalization
- Category assignment
- Dashboard integration

## Setup

```bash
npm install
npm run dev
